Introduce the tots to the world of programming using the popular visual block programming tool Scratch Jr.

Age 5+

Introduce kids to the world of logical thinking and reasoning by using a free intuitive visual block programming tool.

Snapshot

 Children snap together graphical programming blocks to make characters move, jump, dance, sing, add their own voices and sounds. Use programming blocks to make characters come to life. 

Tools Used

Scratch Jr. is an Introudctory programming language app developed in collaboration with the MIT Media Lab that enables young children to create their own interactive stories and games.

Skills Learnt

They learn logical and reasoning concepts like sequencing, loops, conditionals, messaging etc. Develop a personalized constructive game that brings their imagination to reality.
