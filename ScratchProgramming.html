Scratch Programming

Learn the amazing and exciting ideas behind computational thinking and coding language concepts.

Age 5+

Scratch makes programming fun and intuitive, yet is extensible and powerful enough to support general computation. 

Snapshot

Explore Scratch's awesome interface. and discover how to create vector and bitmap art in Scratch's paint editor. Animate and make art interactive with Scratch's extremely helpful puzzle-piece code blocks that make learning how to program fun and easy

Tools Used

Learn the amazing and free-to-use Scratch programming environment, developed in collaboration with the MIT Media Lab that enables young children to create their own interactive stories and games.

Skills Learnt

They will know basic coding concepts and have basic programming skills to use as a launch pad for learning more advanced programming languages
