AR 3D Games

3D Augmented Reality game development 

Age 5+

Take programming to the next level and develop a 3D game with AR interactivity.

Snapshot

Learn about AR, how to enhance the reality, AR triggers and combine with Game ideas to build a 3D game while learning core programming and logic concepts. 

Tools Used

Unity is one of the most commonly used game engines and game editors in the game industry, in education, and by hobbiest game makers today.

Skills Learnt

Create and position game objects, apply scripts, create and apply materials, add lighting and audio, adjust cameras, build the project, and package the project
